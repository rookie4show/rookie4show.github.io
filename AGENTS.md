# AGENTS.md

## 目录定位

`rookie4show.github.io` 是最终静态站点输出 / 发布目录。

通常情况下，这里应视为生成产物，而不是主要开发入口。

## 工作原则

- 默认不要直接在这里修改页面、脚本、样式或资源
- 若最终页面结果有问题，优先回到源码目录排查：
  - 博客结构或主题问题：`rookie4show.github.io.private`
  - 专题页面问题：`ShowCases/<topic-slug>`
  - 共享演示能力问题：`ShowEngine`
- 只有在用户明确要求直接修补发布产物时，才在这里做定向修改

## 核心提醒

如果这里的内容与源码目录不一致，以源码目录和重新生成流程为准。
